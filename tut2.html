Operators In C++

    Operators are symbols that perform operations on variables and values.

    Operators are used for producing output by performing various types of calculations on two or more inputs.

  1. Arithmetic Operators
  2. Assignment Operators
  3. Relational Operators
  4. Logical Operators
1] C++ Arithmetic Operators

Arithmetic operators are used for performing mathematical operations on variables and data. like + ,   - , * ...etc.

Operator    Operation
  +     Addition
  -     Substraction
  *     Multiplication
  /      Division
  %     Modulo Operation (Remainder after division)



Let's See With Some Examples
  // Arithmetic operators
#include<iostream>
using namespace std;

int main(){
    int a=9, b=2; 
  //the sum of a and b.
    cout<<"The value of a + b is "<<a+b<<endl;
  //the difference of a and b.
    cout<<"The value of a - b is "<<a-b<<endl;
  //the product of a and b.
    cout<<"The value of a * b is "<<a*b<<endl;
  //the division of a by b
    cout<<"The value of a / b is "<<a/b<<endl;
  //the modulo of a by b
    cout<<"The value of a % b is "<<a%b<<endl; 
    return 0;
}
Program Output 
The value of a + b is 11
The value of a - b is 7
The value of a * b is 18
The value of a / b is 4
The value of a % b is 1

Process finished.

/ Division Operator

Note the operation (a / b) in our program. The  /  operator is the division operator.

As we can see from the above example, if an integer is divided by another integer, we will get the quotient. However, if either divisor or dividend is a floating-point number, we will get the result in decimals.

 In C++,

 9/2 is 3
 9.0 / 2 is 4.5
 9 / 2.0 is 4.5
 9.0 / 2.0 is 4.5

% Modulo Operator

The modulo operator % computes the remainder. When a = 9 is divided by b = 2 , the remainder is 1.

    Note: The % operator can only be used with integers.




2] C++ Assignment Operators

assignment operators are used to assign values to variables. 

for example:  
    int a = 34;  // a is assigned 34.
  Operator      Example       Equivalent to
  =   a = b;   a = b;  
  +=   a += b;   a = a + b; 
  -=   a -= b;   a = a - b; 
  *=   a *= b   a = a * b; 
  /=   a /= b;   a = a / b; 
  %=   a %= b;   a = a % b; 


Example
#include <iostream>
using namespace std;

int main() {
    int a, b;

    a = 2;    // a is assigned 2
  
    b = 7;    // b is assigned 6

    cout << "a = " << a << endl;
    cout << "b = " << b << endl;
    cout << "\nAfter a += b;" << endl;

    // assigning the sum of a and b to a
    a += b;  // a = a + b
    cout << "a = " << a << endl;

    return 0;
}
Program Output
 a = 2
 b = 7

 After a += b;
 a = 9


3] C++ Relational Operators

A relational operator is used to check the relationship between two operands. 

 example:
   // checks if a is ber than b
 a < b;

Here,  >  is a relational operator. It checks if a is greater than b or not.

If the relation is true, it returns 1 whereas if the relation is false, it returns 0.

  Operator     Meaning
==   Equals operator
!=   Not Equals operator
<   ber than
>    Greater Than
<=   Less Than or Equal To
>=    Greater Than or Equal To


Example
#include <iostream>
using namespace std;

int main() {
    int a, b;
    a = 3;
    b = 5;
    bool result;

    result = (a == b);   // false
    cout << "3 == 5 is " << result << endl;

    result = (a != b);  // true
    cout << "3 != 5 is " << result << endl;

    result = a > b;   // false
    cout << "3 > 5 is " << result << endl;

    result = a < b;   // true
    cout << "3 < 5 is " << result << endl;

    result = a >= b;  // false
    cout << "3 >= 5 is " << result << endl;

    result = a <= b;  // true
    cout << "3 <= 5 is " << result << endl;

    return 0;
}
program output 
 3 == 5 is 0
 3 != 5 is 1
 3 > 5 is 0
 3 < 5 is 1
 3 >= 5 is 0
 3 <= 5 is 1

    Note: Relational operators are used in decision-making and loops..



4] C++ Logical Operators

Logical operators are used for comparing two expressions. For example ((a==b) && (a>b)).

Also logical operators are used to check whether an expression is true or false. If the expression is true, it returns 1 whereas if the expression is false, it returns 0.

Operator     Meaning
  &&     Logical AND. True only if all the operands are true.
  ||     Logical OR. True if at least one of the operands is true.
  !     Logical NOT. True only if the operand is false.
  /      Division
  %     Modulo Operation (Remainder after division)



Example
#include <iostream>
using namespace std;

int main() {
    bool result;

    result = (4 != 5) && (4 < 5);     // true
    cout << "(4 != 5) && (4 < 5) is " << result << endl;

    result = (4 == 5) && (4 < 5);    // false
    cout << "(4 == 5) && (4 < 5) is " << result << endl;

    result = (4 == 5) && (4 > 5);    // false
    cout << "(4 == 5) && (4 > 5) is " << result << endl;

    result = (4 != 5) || (4 < 5);    // true
    cout << "(4 != 5) || (4 < 5) is " << result << endl;

    result = (4 != 5) || (4 > 5);    // true
    cout << "(4 != 5) || (4 > 5) is " << result << endl;

    result = (4 == 5) || (4 > 5);    // false
    cout << "(4 == 5) || (4 > 5) is " << result << endl;

    result = !(5 == 2);    // true
    cout << "!(5 == 2) is " << result << endl;

    result = !(5 == 5);    // false
    cout << "!(5 == 5) is " << result << endl;

    return 0;
}

program output 
(4 != 5) && (4 < 5) is 1
(4 == 5) && (4 < 5) is 0
(4 == 5) && (4 > 5) is 0
(4 != 5) || (4 < 5) is 1
(4 != 5) || (4 > 5) is 1
(4 == 5) || (4 > 5) is 0
!(5 == 2) is 1
!(5 == 5) is 0
Explanation of logical operator program
  • (4 != 5) && (4 < 5) evaluates to 1 because both operands (4 != 5) and (4 < 5) are 1 (true).
  • (4 == 5) && (4 < 5) evaluates to 0 because the operand (4 == 5) is 0 (false).
  • (4 == 5) && (4 > 5) evaluates to 0 because both operands (4 == 5) and (4 > 5) are 0 (false).
  • (4 != 5) || (4 < 5) evaluates to 1 because both operands (4 != 5) and (4 < 5) are 1 (true).
  • (4 != 5) || (4 > 5) evaluates to 1 because the operand (4 != 5) is 1 (true).
  • (4 == 5) || (4 > 5) evaluates to 0 because both operands (4 == 5) and (4 > 5) are 0 (false).
  • !(5 == 2) evaluates to 1 because the operand (5 == 2) is 0 (false).
  • !(5 == 5) evaluates to 0 because the operand (5 == 5) is 1 (true).
